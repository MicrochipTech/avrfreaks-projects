techFX reflow tools
------------------

System requirements:

NET framework 2.0
Windows XP SP2 x32, x64
Windows Vista x32, x64

(if you run the program and get an error instantly, then please do a search for net 2.0 and download and install the update)

USB 2.0 PORT and cable.

AC/DC adapter for controller (will not run on USB power!!)


If running UAV, right click on icon, and select run as administrator.


---------------------

you may have to disable the power savings feature of the driver in the control panel.

goto control panel with controller plugged in, then properties of "pic18f4550 family device"
then power management tab, and disable feature.
