techFX reflow tools
------------------

System requirements:

Windows XP with NET 2.0 framework installed.

(if you run the program and get an error instantly, then please do a search for net 2.0 and download and install the update)

USB 2.0 PORT and cable.

AC/DC adapter for controller (will not run on USB power!!)



---------------------

Windows Vista:

The software has been tested on windows vista, however to run it UAC mode, please right click on the program and run it as administrator in "winXP SP2 compatability mode"


----------------------


---------------------
