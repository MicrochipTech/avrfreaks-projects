# WebPort

Uploaded by haraldkipp on 2002-07-14 19:51:00 (rating 0 out of 5)

## Summary

This application demonstrates how to create a simple web interface to monitor and control I/O ports of an embedded system via TCP/IP by using a browser.


An Ethernut Board with programming adapter or similar hardware is required to run the binary, which is supplied in Intel hex format for ATmega 103 and ATmega 128. Schematics and board layouts are available at [www.ethernut.de](http://www.ethernut.de/).


To modify and compile the included source code, you need the Nut/OS libraries version 2.5 or above and the GNU Compiler Collection version 3.2 for AVR.
