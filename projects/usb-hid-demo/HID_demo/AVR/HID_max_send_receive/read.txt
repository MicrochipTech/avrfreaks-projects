This example demonstrate how to send and receive data from to MCU by USB
Using by PC software and user by defined pin change.
Maximum transfer data 64bytes
Use:
GetReport	- by user software
SetReport	- by user software
WriteData	- by pin change 

Used internal RC oscilator 8Mhz
Tested in ARDUINO LEONARDO dev board.
Powered from USB.
UART out used for debug data..

This fimrware don't use Report ID
Final source code is less then 1500 bytes falash and 68 bytes ram.
