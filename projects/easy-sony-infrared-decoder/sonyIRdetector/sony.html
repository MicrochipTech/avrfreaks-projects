Technical Info

Code length:	12 bits (15 bits for some videocamera functions)
Carrier:		40kHz

	     +--------+
Header:      |        |
	     +        +--+..
		 4T    T

	     +----+
1 is coded:  |    |
	   ..+    +--+..
	       2T  T

	     +--+
0 is coded:  |  |
	   ..+  +--+..
	      T  T

T = 550us approx

Space between data: 25ms

data: hhhhxxxxxxyyyyyy
	  ^          ^
	  MSB        LSB

xxxxxx command
yyyyyy address ( the number of the apparatus )

tape deck a    # 20
tape deck b    # 1C
dat       # 38
cd        # 22
