About ButterflyConnect

ButterflyConnect is an application used to manage the song book of an ATMEL's Butterfly board loaded with the ButterflyAlarm firmware. The user can download new songs and delete the existing ones.

Pre-requisitives

To be able to run ButterflyConnect, you need the following:

  • Python 2.2 or greater
  • wxPython
  • pySerial

If you want to create a Windows executable you need:

  • py2exe

Connections

You can connect with the hardware through a Serial Port in your PC. The required hardware connections are described in the AVR Butterfly Evaluation Kit User Guide. After you make the connections, you only need to select the number of the Serial Port and press the Connect button. The other settings are fixed to 9600, 1 stop bit, no parity, no handshake. If the Butterfly board is detected, you should see a line with the firmware's version number in the status bar.

Editing Songs

Local songs are stored in the Songs folder. The distribution of the program comes with some ready-made songs. If you add or remove a song in the Songs folder, you must press the Update button to see the changes appear.

Local songs are stored in ASCII text files (.txt extension) in a format the application can understand. The first line of the file is the song's name. The maximum size for the song is 12. Names longer than 12 characters are trunctated. The second line should contain a number between 1 and 15, which denotes the tempo of the song. The third line could be either Repeat or No repeat. After that 1 up to 100 lines follow. This is the song's body, which contains one note per line. A note has the following format:

Duration,Tone

Actions

  • To see the songs currently loaded in the Butterfly, press the button Refresh under the Remote Songs list
  • To delete a song in the Butterfly Song Book, select it and press the Delete button
  • To download a local song to the Butterfly, select it and press the Download Button
