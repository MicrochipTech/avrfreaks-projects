ButterflyConnect readme.txt

To start the program run the ButterflyConnect.exe executable.


ButterflyConnect is a wxPython application used to communicate with an Atmel's
Butterfly Evaluation Board loaded with ButterflyAlarm firmware. To be able to
run it, you need the following:

- Python 2.2 or greater
- wxPython (http://wxPython.org/)
- pySerial (http://pyserial.sourceforge.net/)

The Windows executable is created with:

- py2exe (http://www.py2exe.org/)
