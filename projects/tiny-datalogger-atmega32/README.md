# Tiny datalogger on ATMEGA32

Uploaded by andrebrussee on 2007-03-22 18:32:00 (rating 0 out of 5)

## Summary

Features:


- Software clock  

- Adjustable sample rate (1 sec - 255 sec)  

- Ajustable number of channels (1 - 8)  

- Snapshot mode, trigger via PC command  

- Timer mode, trigger via the clock  

- Output in delimited ASCII  

- PC menu via terminal emulation  

- Presets save in memory

## Compilers

- AVR Assembler
