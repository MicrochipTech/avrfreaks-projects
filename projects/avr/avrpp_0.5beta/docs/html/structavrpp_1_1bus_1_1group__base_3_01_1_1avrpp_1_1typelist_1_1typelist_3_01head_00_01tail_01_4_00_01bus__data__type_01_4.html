Detailed Description

template<typename head, typename tail, typename bus_data_type>
struct avrpp::bus::group_base< ::avrpp::typelist::typelist< head, tail >, bus_data_type >

A class optimizing group operations.

This class should be instantiated with a list of bus_pin classes, each of which should share the same value of bit_shift.
