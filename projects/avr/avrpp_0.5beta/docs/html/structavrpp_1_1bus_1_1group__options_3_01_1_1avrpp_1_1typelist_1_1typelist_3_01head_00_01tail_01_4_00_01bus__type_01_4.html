Detailed Description

template<typename head, typename tail, typename bus_type>
struct avrpp::bus::group_options< ::avrpp::typelist::typelist< head, tail >, bus_type >

A class optimizing setting options for similar pins.

This class should be instantiated with a list of bus_pin classes, each of which should share the same options type.
