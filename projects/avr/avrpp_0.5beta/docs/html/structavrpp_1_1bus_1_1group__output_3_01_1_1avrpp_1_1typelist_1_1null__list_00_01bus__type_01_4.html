Detailed Description

template<typename bus_type>
struct avrpp::bus::group_output< ::avrpp::typelist::null_list, bus_type >

A class optimizing group output.

This class should be instantiated with a list of bus_pin classes, each of which should have the same type of "bulk" nested typedef and the same value of bit_shift.
