Detailed Description

template<typename head, typename tail, typename bus_type>
struct avrpp::bus::group_output< ::avrpp::typelist::typelist< head, tail >, bus_type >

A class optimizing group output and config.

This class should be instantiated with a list of bus_pin classes, each of which should share the same type of "bulk" nested typedef and the same value of bit_shift.
