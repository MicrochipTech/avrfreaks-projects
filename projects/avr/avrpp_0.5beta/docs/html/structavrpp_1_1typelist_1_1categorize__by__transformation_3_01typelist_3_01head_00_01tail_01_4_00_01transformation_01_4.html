Detailed Description

template<typename head, typename tail, template< typename source > class transformation>
struct avrpp::typelist::categorize_by_transformation< typelist< head, tail >, transformation >

Splits a type list into categories defined by transformation.

Performs a type transformation (see transform<>), then creates a list of lists. Every sublist contains types that are transformed to same type. Classes from different sublists are always transformed to different types.

Parameters:
class_list type list to categorize
transformation is a template class parameterized by a class being transformed and producing a new (transformed) class
Returns:
result is type list of type lists, one for each category
