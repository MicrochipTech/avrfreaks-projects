Detailed Description

Command system

This provides command functionality for a terminal interface A list of commands is defined by the user at compile time, which is passed to the initialization routine at run time.

"Command line" strings are passed to the CMD::FindAndRun() routine which then parses the command, searches for a match, and runs the callback routine.
