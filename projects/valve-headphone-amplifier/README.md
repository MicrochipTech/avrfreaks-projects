# Valve headphone amplifier

Uploaded by atkins on 2009-06-25 12:14:00 (rating 0 out of 5)

## Summary

Valve headphone amplifier with transistor output stage. Using a dual triode as a preamp and class a transistor stage as output driver. Powered from 12V, using a Royer converter to produce +200Vdc and 6.3Vdc for Valve HT and heaters. Also uses EM84 'magic eye' as an output level indicator. ATtiny24 controls and monitors power supply, providing heat up times and power sequencing. Link to page [here](http://www.nanobox.endoftheinternet.org/Download/files/Head/head.html)
