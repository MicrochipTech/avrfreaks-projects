Compiling with Linux

  • log on as user cc (home directory will be /home/cc)
  • Have the DOS/Windows source files in /C/home/cc/... 
    cp /C/home/cc/profile.bin ~/.bashrc
    log off and log on again 
    mkdir ~/linux; cp /C/home/cc/linux/* ~/linux
mkdir ~/bin; cp /C/home/cc/bin/* ~/bin
mkdir ~/include; cp /C/home/cc/include/* ~/include
mkdir ~/lq; cp /C/home/cc/lq/* ~/lq
mkdir ~/util; cp /C/home/cc/util/* ~/util
mkdir ~/qq; cp /C/home/cc/qq/* ~/qq
chmod ugo+x ~/linux/*
chmod ugo+x ~/bin/*
chmod ugo+x ~/lq/tl
chmod ugo+x ~/lq/tll
  • check if all file names are in lower cases
  • compile the library 
    cd ~/lq
ccsif asctosl
tl
  • Compile, link and run a test program (qqmain) 
    cd ~qq
g++ qqmain.cpp
cp a.out ~/bin
a.out
  • Compile and link syncdir 
    cd ~/util
ccs syncdir
lnk syncdir
  • Changes have been neccesary? Copy them back. Save syncdir, remove all sources 
    rm lq
rm util
rm qq
rm include
  • Run the saved syncdir 
    syncdir -12 -r ~ /C/home/cc
