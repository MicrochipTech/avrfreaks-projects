Compilation with MSVC 1.0 or 1.52b

  1. The Directory C:\HOME\CC\include must be included in include paths (Menu [Options] [Directories] )
  2. For each main program there should be a .mak file (a Cpp project) in subdirectory UTIL/MSVC10 or in QQ/MSVC10
  3. When Creating a new project do not [ ] Use Microsoft Foundation Classes
  4. When compiling from command line, then the files from \home\cc\mdos10.b are used:
    • ccs.bat to compile a single source
    • lnk.bat to link to .exe
Key strokes :
  • Shift-F8 : to Build a Project [Project] [Build]
