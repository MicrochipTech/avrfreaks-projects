## Summary

This is a compilation of library's an drivers dedicated for ATxmega A microcontrollers.


This compilation include the next lib's and drivers:


RTC driver.  

DMA driver.  

SPI driver.  

Usart driver.  

System clk driver.  

IO macro driver.  

Math lib.  

Utility lib.
