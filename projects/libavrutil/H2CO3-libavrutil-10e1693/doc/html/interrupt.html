libavrutil - Interrupts

AVR microcontroller untis provide two or eight (on the ATmega1280 and ATmega2560) external interrupts. Interrupt handler functions can be set, to be called when an enabled external interrpt occurs. Refer for your AVR's datasheet to find out on which ports are the external interrupt pins are located. For these functions to work, global interrupts should be enabled.

AVR_INTR_EXT_0, 1, 2, 3, 4, 5, 6, 7: the external interrupt number constants.

AVR_INTR_MODE_CHANGE, AVR_INTR_MODE_FALLING, AVR_INTR_MODE_RISING: specifies when an interrupt should be fired: on pin level change (any directon, either from high to low or from low to high), on a signals failing transition edge (from high to low) or on a rising edge (from low to high).

avr_interrupt_enable(): equivalent to sei().

avr_interrupt_disable(): equivalent to cli().

void (*avr_interrupt_handler)(): interrupt handler function prototype; all interrupt handler functions should conform to the type void interrupt_handler().

void avr_interrupt_set_handler(uint8_t intnum, avr_interrupt_handler hndl, uint8_t mode): enables the external interrupt for number intnum, calling hndl when that particular interrupt occurs.

void avr_interrupt_remove_handler(uint8_t intnum): disables the external intterupt intnum and removes its handler function.
