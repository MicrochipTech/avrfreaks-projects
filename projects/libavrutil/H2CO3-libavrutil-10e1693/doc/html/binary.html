libavrutil - Binary operations

Binary operations are fairly easy to perform. Libavrutil provides a few macros for bitwise operations. These macros, when applied to the AVR's appropriate ports, provide data direction (in/out) control, digital output and digital input.

avr_bit_set(value, bit): sets the bit bit of the byte/word/long word value to one.

avr_bit_clear(value, bit): clears the bit bit of the byte/word/long word value (sets it to 0).

avr_bit_write(value, bit, isset): sets or clears the bit bit of the byte/word/long word value depending on isset being 1 or 0, respectively.

avr_bit_toggle(value, bit): inverts the bit bit of the byte/word/long word value

avr_bit_isset(value, bit): Returns nonzero if the bit bit of the byte/word/long word value is set.

avr_bit_isclear(value, bit): Returns nonzero if the bit bit of the byte/word/long word value is clear.

B00000000 to B11111111: Definitions of one-byte numbers, from 0x00 to 0xff, respectively, to make bitmask operations look more obvious.

byte_t: 8-bit (one byte) unsigned integer type.

u_byte_t: Alias for byte_t

s_byte_t: 8-bit (one byte) signed integer type.
