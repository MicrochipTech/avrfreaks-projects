libavrutil - Pulses

Generating straight or inverted logic pulses of microsecond and millisecond precision time intervals is possible using the following functions (naming is self-explanatory):

void avr_pulse_generate_us(uint8_t *port, uint8_t pin, uint16_t us)

void avr_pulse_generate_ms(uint8_t *port, uint8_t pin, uint32_t ms)

void avr_pulse_generate_inv_us(uint8_t *port, uint8_t pin, uint16_t us)

void avr_pulse_generate_inv_ms(uint8_t *port, uint8_t pin, uint32_t ms)

Measuring length of pulses is also possible using the following functions:

uint32_t avr_pulse_measure_us(uint8_t *port, uint8_t pin)

uint32_t avr_pulse_measure_ms(uint8_t *port, uint8_t pin)

uint32_t avr_pulse_measure_inv_us(uint8_t *port, uint8_t pin)

uint32_t avr_pulse_measure_inv_ms(uint8_t *port, uint8_t pin)

Microsecond precision measure functions may not be precise; see the notice in the pulse.h header file. The measure functions should be called a few microseconds earlier than the start of the actual pulse in order them to be able to do some internal setup.
