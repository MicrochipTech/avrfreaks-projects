libavrutil documentation

libavrutil is a library for the AVR platform. It's intended to provide a bunch of useful utility functions for easily performing common microcontroller tasks, such as A/D and D/A conversion, servo control, audio output, etc.

The library is written in C, and relies heavily on the standard GNU AVR toolchain (gcc-avr and avr-libc). It can be built on almost any Unix-like system (Linux, Mac OS X...) without modification, and can be easily adapted to Windows (by modifying the Makefiles).
