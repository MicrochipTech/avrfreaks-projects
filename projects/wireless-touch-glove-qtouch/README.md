# Wireless Touch Glove with Qtouch

Uploaded by bankole.peter on 2013-03-11 04:19:00 (rating 0 out of 5)

## Summary

This Package includes videos, tutorials, code, and presentation for the Wireless Touch Glove. The Wireless Glove works just like the Peregrine Glove used for PC games. When you touch certain part of the glove it sends data to the the PC with a Bluetooth module. The data can be monitored using a terminal emulator. The project can be found at [http://spaces.atmel.com/gf/proje...](http://spaces.atmel.com/gf/project/wireless_glove/docman/)
